# Movie List App (iOS)

An iOS application that displays a list of movies using a UITableView.  
Each movie shows an SF Symbol icon, title, vote count, and release year.  
Selecting a movie navigates to a detail screen.

## Features

- UITableView-based movie list
- SF Symbols for icons
- Navigation using UINavigationController
- Custom table view cells
- Clean and simple UI
- Safe optional handling (no Optional(...) shown)

## Tech Stack

- Swift
- UIKit
- UITableView
- SF Symbols
- MVC Architecture
- iOS 14+

## App Flow

1. Launch app
2. Movie list is displayed
3. Tap on a movie cell
4. Navigate to the next screen with details
