> **The app itself is built entirely programmatically (no Storyboards or XIBs).**

---

## **Overview**
An iOS application that displays a list of movies in a clean, simple UI using a `UITableView`. Each movie cell shows an SF Symbol icon, title, vote count, and release year. Selecting a movie navigates to a detail screen.

---

## **Features**
- **Programmatic UI** – Built entirely in Swift code without Storyboards or XIBs.  
- **UITableView-based movie list** – Scrollable list of movies.  
- **SF Symbols** – Each movie cell includes an icon.  
- **Navigation** – Uses `UINavigationController` for smooth screen transitions.  
- **Custom Table View Cells** – Displays icon, title, votes, and year clearly.  
- **Safe Optional Handling** – Prevents "Optional(...)" from appearing in the UI.  
- **Clean and simple UI** – Minimalist, user-friendly design.  

---

## **App Flow**
1. Launch the app.  
2. The movie list is displayed in a `UITableView`.  
3. Tap on a movie cell.  
4. Navigate to a detail screen showing more information about the selected movie.  

---

## **Tech Stack**
- **Language:** Swift  
- **UI Framework:** UIKit  
- **Views:** UITableView, UITableViewCell, UINavigationController  
- **Icons:** SF Symbols  
- **Architecture:** MVC  
- **Minimum iOS Version:** iOS 14+
